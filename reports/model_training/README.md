# reports/model_training/

Visualizaciones generadas por `model_training_evaluation.py`:

- `tabla_comparativa_modelos.csv` — métricas comparativas de los 4 modelos entrenados.
- `roc_curve_comparativa.png` — curvas ROC de los 4 modelos.
- `precision_recall_curve.png` — curvas Precision-Recall comparativas.
- `matriz_confusion_<modelo>_<threshold>.png` — matrices de confusión por modelo y umbral.
- `feature_importance.png` — importancia de variables del modelo ganador.
- `curva_costo_vs_threshold.png` — análisis del threshold tuning.

Estos gráficos documentan el proceso de selección y optimización del modelo.
